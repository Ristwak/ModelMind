# ЁЯза Outsmart the Output (ModelMind)

**Outsmart the Output** is an educational game built in Unity for postgraduate students to improve their AI/ML prompt engineering skills. The player is given realistic scenarios and must select the most effective prompt for an AI model to solve the given task тАФ all under time pressure.

---

## ЁЯОп Game Concept

- Each question presents a **real-world use case or problem** in Hindi.
- The player sees **three prompt options** that are very similar and closely worded.
- Only one prompt is perfectly tuned to produce the desired AI output.
- The player must choose the best prompt within the time limit.

- тП│ Timed rounds (auto-submit on timeout)
